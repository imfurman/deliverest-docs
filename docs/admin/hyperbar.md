# Гипербар: быстрые действия в админ-панели

Гипербар помогает менеджерам и маркетологам быстро находить страницы админ-панели, товары и справочники из одной строки поиска. Это сокращает путь до нужного раздела и избавляет от ручного перехода по меню. Вызовите панель двойным нажатием `Ctrl`.

## Возможности

- Быстрый переход на страницы админ-панели, отчёты и служебные разделы без навигации по меню.
- Поиск позиций каталога с ценой и разделом, чтобы оперативно проверить карточку товара.
- Подсказки по разделам каталога, городам и цехам для проверки настроек или открытие карточек.
- Поиск по номеру телефона для быстрого перехода к контактам клиента.
- Модуль списков (города и цеха) — готовые подборки по ключевым словам прямо в поисковой строке.

### Модуль списков: города и цеха

- **Зачем нужен.** Открывает полный список городов или цехов без ручного поиска по меню. Удобно, когда нужно проверить настройки региона или быстро переключиться между цехами.
- **Как работает.** Введите в гипербар «город»/«города» или «цех»/«цеха». Появится список актуальных сущностей с подсказкой по принадлежности (для цехов отображается город). Клик по результату ведёт сразу в карточку редактирования.
- **Какие задачи закрывает.** Быстрая проверка атрибутов города и цеха, переход к корректировке расписания или зон доставки, навигация по нескольким площадкам в один клик.
- **Кому доступно.** Доступен пользователям с правами админ-панели (маркет, администраторы, супер-администраторы) — права проверяются так же, как и для остальных модулей гипербара.

## Как использовать гипербар

1. Откройте гипербар в админ-панели двойным нажатием `Ctrl`.
2. Начните вводить запрос: название страницы, телефона, позиции каталога или ключевые слова «город»/«цех» для готовых списков.
3. Выберите подходящий результат и переходите к редактированию или просмотру без дополнительных кликов по меню.
