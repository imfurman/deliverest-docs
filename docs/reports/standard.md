# Стандартные отчёты

Раздел «Стандартные» объединяет отчёты для ежедневного мониторинга продаж, маркетинга и финансов. Ниже — самые востребованные группы с подсказками, как их использовать.

## Продажи и выручка

- **Цехи** - Показывает количество заказов, суммы по видам оплаты, долю онлайн-оплаты и бонусов. Используйте, чтобы сравнивать смены и выявлять цехи, которые проседают по обороту.
- **Цехи по дням** - Структурирует оборот по датам и времени суток, помогает анализировать выходные, праздничные пики и ночные смены.
- **Заказы по цеху** - Содержит каждый заказ с адресом, суммой и типом оплаты. Подходит для сверки с инкассацией и разбора спорных случаев.
- **Продажи по часам** - Даёт тепловую карту спроса (шаг 15 минут). Сравните будни и выходные, чтобы адаптировать график операторов и кухонь.
- **Самовывоз** - Сводит в таблицу количество и сумму самовывозов по цехам. Удобен для оценки влияния акций на take-away.

## Каталог и ассортимент

- **Продажи по позициям** - Показывает объём продаж и выручку по позициям, разделяет наличные и безналичные оплаты. Фильтры по цеху и разделу помогут увидеть, какие блюда стоит продвигать в конкретной локации.
- **Продажи по позициям — выгрузка в 1С** - Содержит данные по каждому заказу: цена, скидка, бонусы, контакт клиента. Используйте для передачи в бухгалтерию или внешнюю ERP.
- **Справочник позиций** - Набор позиций с привязкой к разделам. Полезно перед обновлениями каталога или аудитом меню.
- **Категории позиций** - Определяет долю от оборота для групп меню (роллы, пицца, напитки). Помогает понять, какие разделы растут или требуют промо.

## Маркетинг и клиенты

- **Заказы по платформам** - Сравнивает сайт, приложение, агрегаторов и call-центр. Отслеживайте, как влияют рекламные кампании или изменения в приложении.
- **Источники клиентов** - Показывает, откуда пришёл пользователь, и позволяет оценить эффективность SEO, рекламы и рекомендаций.
- **Скидки** - Суммирует применённые скидки и сравнивает с общей выручкой. Быстро показывает, какие акции приносят наибольший оборот.
- **Детализация скидок** - Раскрывает конкретные заказы с промокодами и ручными корректировками. Используйте для аудита ручных скидок или проверки новых промо.
- **Персональные предложения** - Отслеживает отправки и погашения персональных купонов, даёт дату последнего использования.
- **Постоянные клиенты** - Помогает оценить, как быстро клиенты возвращаются и сколько заказов успевают сделать.
- **Бонусные уровни** - Показывает, сколько клиентов попало в каждый уровень за период. Помогает регулировать условия программы лояльности.
- **Заказы по тегам** - Следите за VIP-гостями, техническими проблемами или тестовыми заказами. Теги удобно использовать как чек-лист для менеджеров.
- **Push-рассылки** - Видно, сколько уведомлений доставлено, какая ОС и сколько персональных ссылок использовали.
- **Сбор email** - Быстрая выгрузка свежих контактов, собранных через формы подписки.
- **Отзывы клиентов** - Отчёт содержит оценки оператора, курьера, кухни и текст комментария, что помогает обнаружить болевые точки сервиса.

## Финансы и контроль

- **Сверка с АТОЛ** - Сравнивает суммы заказов с фискальными документами, что позволяет оперативно найти несоответствия.
- **Неоплаченные заказы** - Фиксирует заказы в статусе «не оплачен» вместе с данными платёжного шлюза. Подходит для контроля возвратов и работы службы взыскания.
- **Долги между цехами** - Показывает, какие цеха оплачивают заказы друг друга, и выдаёт итоговую разницу.
- **Детализация долгов по заказам** - Раскрывает конкретные заказы, из-за которых возникли взаиморасчёты. Используйте перед закрытием месяца.
- **Инкассация курьеров** - Отображает, сколько наличных и по карте принёс каждый курьер; подходит и для бухгалтерии.
- **Лог облачной печати** - Помогает контролировать печать чеков и заказов — пригодится, если возникают претензии по отсутствующим фискальным документам.

## Работа с фильтрами и экспортом

- Фильтр по датам поддерживает быстрые пресеты: «вчера», «неделя», «со смены по смену». Для глубокого анализа вводите дату и время вручную.
- Дополнительные селекторы (цех, оператор, тип скидки) влияют не только на таблицу, но и на экспорт. Это удобно, когда нужно отправить в бухгалтерию данные по одному подразделению.
- Большинство отчётов позволяют добавить в «Избранное» — используйте это для ежедневных проверок.
- Если отчёт предполагает быструю сверку данных (например, «Заказы по цеху» + «Инкассация курьеров»), откройте их в отдельных вкладках и сравнивайте прямо в браузере, не скачивая файлы.
