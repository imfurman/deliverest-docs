- Добавили метод, через который курьер может поменять тип оплаты заказа: с наличных на карту курьеру и обратно.
- Поставили защиту от ошибок: менять можно только свой заказ, только в рабочих статусах и не для онлайн-оплаты.
- Это снизило количество ручных правок через операторов и ускорило закрытие заказов в поле.
