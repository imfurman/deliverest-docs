- В настройках цеха для Атол появились поля «Принципал (юрлицо цеха)» и режим работы «только как принципал» — теперь продавец задаётся на уровне конкретного цеха.
- При формировании чека данные принципала подставляются для товарных позиций, а центральная компания может выступать агентом по приёму платежей.
- Это разделяет выручку между цехами, снижает риск налоговых расхождений и делает чеки юридически корректными.
- Настройку можно включать точечно для нужных цехов, без изменения остальных.
