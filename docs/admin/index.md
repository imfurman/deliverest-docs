# Инструменты

В этом разделе собраны инструменты, которые помогают администраторам и маркетологам быстро работать в системе: быстрые действия, навигация и вспомогательные панели.

- [Гипербар](hyperbar.md) — поиск страниц, товаров, городов и цехов из одной строки, открывается двойным `Ctrl`.
- [MasterBot](../masterbot/index.md) — Telegram-бот для уведомлений и быстрых команд без входа в админ-панель.
