- При создании заказа через mobile API v2 проверяется, совпадает ли телефон контакта с `username` пользователя; при расхождении запрос отклоняется.
- Добавлена валидация в контроллер, модель и менеджер заказов, чтобы исключить неправильные номера и дубли пользователей.
